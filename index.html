Projects

pAInt - 2017

Built at McHacks 2017, my team and I developed a lightweight web applications where users can upload and image and stylize it to resemble any of a collection of 9 paintings. We developed code to train a generative convolutional neural network to learn style transformations, but due to time constraints had to use pretrained models found here. I'm currently working on implementing research by Dumoulin, et al. which uses a single trained network to generate images with any of 32 styles.

N Body Simulation - 2016

We took 3rd place at McGill Physics Hackathon with our N-Body gravitational simulation. The core simulation was written in C++ for high performance, then simulation frames were pushed to a web app in real time for visualization. Frames were then buffered and then displayed in full 3D using three.js.
